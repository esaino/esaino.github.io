# esaino.github.io

Personal website and portfolio - Lovable-powered React application.

## Live Site

https://esaino.github.io

## Technology Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions on every push to main.

## Local Development

```bash
npm install
npm run dev
```
